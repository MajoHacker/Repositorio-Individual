¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java y JavaScript son dos lenguajes de programación con algunas similitudes superficiales, como una sintaxis básica similar y el soporte para la programación orientada a objetos. Sin embargo, difieren significativamente en su origen y propósito, con Java siendo un lenguaje orientado a aplicaciones empresariales y de escritorio, compilado y ejecutado en una máquina virtual, mientras que JavaScript es un lenguaje de script orientado a web, interpretado y principalmente ejecutado en navegadores.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate(), getMonth() (nota los meses van de 0-11), getFullYear(), getDay() (van de 0-6), getHours()

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

sort(), reduce(), filter(), map(), forEach()

¿Cómo se declara una variable con alcance local dentro de una función?

Utilizando la palabra clave let o const

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Pueden dificultar la comprensión del código y su mantenimiento, así como si se utilizan nombres de variables comunes, existe el riesgo de colisión de nombres, especialmente cuando hay programas grandes
